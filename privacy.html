#cromulentcofee privacy policy

This webpage constitutes a map of the wold with pins of various locations with good, or hopefully good coffee. The locations are an amalgamation of the "cromulentcoffee" instram API app reading the @cromulentcoffee instagram account. There are further details sources from the cromulentcoffee.github.io account.

No personal or identifying information is input by any users into this system. Users are not asked to log into github nor instagram. It is a read only system.

There is, however, google analytics on this webpage to gauge activity levels on the page.
